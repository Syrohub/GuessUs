# GuessUs — Party Word Game

iOS party game with Family (4+) and Adult (17+) variants.

## Tech Stack

React 18 · TypeScript · Vite 5 · Capacitor 7 · Tailwind CSS 3

## Quick Start

```bash
npm install
npm run dev              # Web dev server at localhost:5173
npx cap sync ios         # Sync to Xcode
npx cap open ios         # Open in Xcode
```

## Build Variants

```bash
./scripts/build-variant.sh family    # Build family version
./scripts/build-variant.sh adult     # Build adult version
./scripts/deploy-all.sh              # Build all
```

## Languages

English, Russian, Spanish, Ukrainian
